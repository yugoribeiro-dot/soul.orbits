# soul.orbits — Brand Kit (v0.1)

## Color palette

| Role | Hex | Name | Usage |
|------|------|------|-------|
| Primary background | `#14122B` | Deep Cosmos | Default post background, dark mode everything |
| Secondary background | `#221E45` | Nebula | Cards, slide layers, gradient stops |
| Primary text | `#F5EFE0` | Moonlight | Headlines, body copy on dark |
| Accent / highlight | `#D4A574` | Old Gold | Symbols, dividers, key words, CTA |
| Muted accent | `#7C6FA8` | Dusk Violet | Secondary text, captions, subtle separators |

**Rules**
- 90% of any post surface is `Deep Cosmos` or `Nebula`
- Gold is a *spice*, not a base — use for one element per slide max
- Never pure white (`#FFF`); always `#F5EFE0`
- Never pure black (`#000`); always `#0B0A1F`
- No gradients except subtle Deep Cosmos → Nebula (vertical, soft)

## Typography

**Free Google Fonts pairing:**

- **Display / Headlines:** [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond) — weight 300 (light) for slide titles, weight 500 (medium) for hooks
  - Why: serif with classical roots, reads "considered" not "trendy"
- **Body / UI:** [Inter](https://fonts.google.com/specimen/Inter) — weight 400 for body, weight 500 for emphasis
  - Why: high legibility at small sizes, neutral, doesn't compete with the serif

**Type rules**
- Headline size: 48–64pt on a 1080x1080 post
- Body size: 22–28pt minimum (readable on a phone)
- Line height: 1.4 for body, 1.2 for headlines
- Letter spacing: +20 on all caps, 0 elsewhere
- ALL CAPS reserved for category labels only ("DAILY TRANSIT", "REFLECTION")
- Never bold the serif; use weight 500 instead of 700

## Iconography
- Astrological glyphs only when accurate and contextual (♀ ☽ ☉ ♃ etc.)
- Source: Unicode astrological symbols, no decorative custom illustrations in v0.1
- One symbol per post max — restraint signals authority

## Voice guidelines

**Do**
- Write at a 9th-grade reading level
- Use concrete sensory language: *"the Moon slips into Pisces tonight"*, not *"lunar transition occurs"*
- Pose questions more than declarations
- Use second person ("you") sparingly — let the reader project
- End on a soft, open beat — never a hype CTA
- Use line breaks like a poem, not paragraphs like an article

**Don't**
- Don't use exclamation marks. Ever.
- Don't say "manifest", "high vibrational", "divine feminine", "5D"
- Don't promise outcomes ("you will…", "this means you'll…")
- Don't stack emojis. One emoji per caption max, ideally zero.
- Don't use astrological jargon without translating it inline
- Don't use "queen", "babe", "soul fam", "tribe"
- Don't be cute. Be quiet.

**Sample phrases — yes**
- *"Mercury turned direct this morning. Notice what wants to be said now."*
- *"The Moon spends today in Scorpio. Old feelings might surface. Let them."*
- *"Saturn doesn't punish. It edits."*
- *"Your North Node points to the room you haven't entered yet."*

**Sample phrases — no**
- ~~*"Babe, this Mercury Retrograde is going to be SO transformative for you ✨💫🌙"*~~
- ~~*"Manifest your dream life under this New Moon energy!"*~~
- ~~*"The universe has BIG plans for you this week, queen"*~~

## Post composition rules
- Generous negative space — at least 30% of the canvas should be empty
- Symmetric or rule-of-thirds composition; never busy
- Text aligned left or center; never right-aligned
- Drop shadows: never. Glows: only on celestial objects (Moon, planets), very subtle.

- Letter spacing +40 to feel celestial
